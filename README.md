# OrangePi_Zero3

How to install WiringPI on OrangePI Zero 3

Step 1 Download:

install_wiringpi.sh

Step 2 Run:

chmod +x install_wiringpi.sh

bash install_wiringpi.sh

Step 3 Run:

sudo python3 led_blink.py
